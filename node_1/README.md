# About

Утилита tree для удобного показа структуры файлов директории.

# Description

## Установка:

```sh
$ npm install -g
```

## Запуск:

```sh
$ tree -p <path> -d <depth>
```

## Параметры

1. '-p, --path <path>', 'Путь до целевой папки (string, required)'
2. '-d, --depth <depth>', 'Опция глубины показа (number, required)'
