# About

Утилита tree для удобного показа структуры файлов директории.

# Description

## Установка:

```sh
$ npm install -g
```

## Запуск:

```sh
$ tree -p <path> -d <depth>
```

## Тесты:

```sh
$ npm run test
```

## Параметры

1. -p, --path: Путь до целевой папки (string, required)
2. -d, --depth: Опция глубины показа (number, required)
